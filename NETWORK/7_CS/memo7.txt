M1 — 기본 HTTP 서버 실행

python -m http.server 8000 
- python이 기본 제공하는 간단한 정적 파일 서버 
- 기본적으로 GET과 HEAD만 지원
- REST API 아님 (파일 서버)

① http:// : 80번 포트가 기본값

② localhost : 자기 자신 컴퓨터 ( localhost = 127.0.0.1 )
-> 서버가 내 컴퓨터에서 실행 중이기 때문에 이렇게 접속합니다.

③ :8000 : 포트 번호
컴퓨터 안에는 여러 프로그램이 동시에 네트워크 통신을 합니다.
이걸 구분하기 위해 “문 번호”가 필요합니다.
이 문 번호가 포트입니다.
